How to Run These Examples

Prerequisites: Install Gemini CLI with Virtual Tools
# Clone and build the special branch with virtual tools support:
git clone https://github.com/EvolvingAgentsLabs/gemini-cli
cd gemini-cli && git checkout issue-1806
# Follow build instructions in the repository

Step 1: Boot LLMunix
# From the agent-examples directory:
./llmunix-boot

Step 2: Start Gemini CLI
gemini
# The GEMINI.md manifest will be automatically detected

Step 3: Execute Example Commands
# Research Intelligence Agent
> Monitor 5 tech news sources, extract trending topics, and generate intelligence briefing

# Legal Contract Analysis
> Review this vendor contract and summarize key terms, risks, and non-standard clauses

# Marketing Campaign Generator
> Create a comprehensive marketing campaign for 'EcoFlow Pro' with budget constraints

# Healthcare Patient Onboarding
> Process new patient intake with HIPAA compliance and comfort detection
