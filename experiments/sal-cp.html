SAL-CP ALPHA

The TCP/IP of agent networks - a standardized protocol for context-aware communication between LLMunix instances running on Gemini CLI. Research into rich messaging standards for distributed AI systems powered by Gemini CLI's runtime engine.

‚ö†Ô∏è Experimental Research - This project explores rich context communication and will remain permanently in alpha status.

üîç Research Problem

In distributed networks of LLMunix agents, simple text messages are insufficient. Agents need to share: cognitive state, resource constraints, task context, capability requirements, and coordination metadata. Without standardization, agent networks cannot scale effectively.

üí° Research Approach: Standardized Agent Messaging

SAL-CP is not a single agent but a message format specification that all LLMunix instances implement for inter-agent communication.

SAL-CP Message = Protocol Version + Sender/Receiver + Payload + Context + Metadata

This creates a common language enabling meaningful collaboration between specialized agents across the network.

üìã Protocol Specification

Research into standardized JSON message format with required fields for protocol version, routing, payload, context, and metadata.

üß† Cognitive State Fields

Standard states like PLANNING, ANALYZING, CREATING, BLOCKED - enabling agents to share their thinking process and resource usage.

üéØ Context Preservation

Research into maintaining task history, shared memory keys, and dependencies as work flows between specialized agents.

üìä Message Types

Standardized types like TASK_REQUEST, BID_SUBMISSION, CONTEXT_SHARE, HANDOFF - enabling rich interaction patterns.

üìö LLMunix Integration

Built into every LLMunix instance's send_message tool, automatically formatting messages to SAL-CP standards.

üîó Network Foundation

Enables EAX Router discovery, Marketplace bidding, and all agent-to-agent coordination in the ecosystem.

Implementation Example

How LLMunix agents use SAL-CP for communication in their virtual tools.

# In LLMunix GEMINI.md virtual tools
def send_message(recipient, content, context=None):
    """Send SAL-CP formatted message to another agent"""
    message = {
        "protocol": "SAL-CP/v1",
        "from": self.agent_id,
        "to": recipient,
        "payload": {"content": content},
        "context": {
            "cognitive_state": self.current_state,
            "confidence_level": self.task_confidence,
            "resource_usage": self.get_resource_metrics()
        }
    }
    return self.message_bus.send(message)
